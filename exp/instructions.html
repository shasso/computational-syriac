How to use Syriac Web Fonts in your Web Site (coming soon...)

  1. You need to include this line in the <head> section of your html page:
    <link href="http://shasso.github.io/sasfonts/earlyaccess/sas-fontstyles.css" rel='stylesheet' type='text/css'>
  2. Create your own css file and include css properties like these:
    font-family: 'east_syriac_marcusregular', 'Noto Sans Syriac Estrangela';
    direction: rtl;
  3. You can include these properties in any style definitions. Here is an example for defining a class named "ES1"
    .ES1 { font-family: 'east_syriac_marcusregular', 'Noto Sans Syriac Estrangela'; direction: rtl; font-size: 2em; }
  4. It is important to use the font-family name exactly as shown because these names are defined in the 'sas-fontstyles.css' file described above
  5. Currently, the following font-faces are defined:
    • 'Assyrian Estrangela' for Dan's Assyrian Estrangela font
    • 'Assyrian-Dan-Regular' for Dan's Assyrian Regular font
    • 'east_syriac_marcusregular' for East Syriac Marcus font
    • 'estrangelo_marcusregular' for Estrangela Marcus font
    • 'aaa-manuscript_marcusregular' for AAA-Manuscript Marcus font
    • 'Noto Sans Syriac Eastern'; for Google's Noto Sans Syriac Eastern font
    • 'Noto Sans Syriac Western' for Google's Noto Sans Syriac Western
    • 'Noto Sans Syriac Estrangela' for Google's Noto Sans Syriac Estrangela font
  6. In your html file, you can style any tag with this class, for example:
    <div class="GS3" > Any Syriac Text </div>
